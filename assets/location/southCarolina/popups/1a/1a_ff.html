Computer Security Office

The Computer Security Office plans, directs, and oversees the implementation of a comprehensive, coordinated, integrated and cost-effective Information Technology (IT) Security program.Â  Its functions include:

  • Ensuring appropriate, effective, and efficient NRC-wide integration, direction and coordination of IT Security planning and performance
  • Being an Agency-level liaison with external entities on mutual IT security interests
  • Formulating and overseeing an IT security program budget
  • Proposing and successfully advocating appropriate Agency-level IT security guidelines
  • Providing vision, leadership, and oversight in developing and promulgating an end-to-end, comprehensive IT security architecture
  • Providing credible, cogent, and timely advice and counsel to the Chairman, Commission, and NRC senior management on aspects of IT Security
  • Adjusting, as appropriate, to meet any evolving threat.
